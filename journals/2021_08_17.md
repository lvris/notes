- 前端模块化
	- [[CommonJS]]
		- `module.exports = {}`
		- `require(<url>)`
	- ES6
		- `export <var>`
			- 支持声明之前定义, 支持多数导出
			- `export default`
		- `import {<var>} from <url>`
			- 可以理解为一种变量解构
			- `import <name> from <url>` | `import {default as <name>} from <url>`
		- `[as y]`
			- 导入导出时均可以使用, 定义别名
			-
- webpack
	- environment
		- 依赖于 node
		- 建议局部安装 `npm install webpack --save-dev`
	- config
		- 配置方法
			- 使用 [[CommonJS]] 导出模块 (见下)
			- default: webpack.config.js
			- separate: `webpackMerge(baseConfig, {separateConfig})`
		- `entry: path`
		- `output: { path, filename }`
		- `module - rules: Array<Object>`
			- `test: RegEx`
			- `use`:
				- `loader: [name: String]`
				- `options`: 使用于特定 loader 的一些选项
		- `resolve`
			- `extensions: Array<String>` (suffix)
			- `alias: { alias: <originName> }`
		- `plugins: Array<Object>`
		- `devServer`
			- `contentBase: string`
			- `port: number`
			- `inline: true` (热刷新)
	- loader
		- 使用: npm 安装之后 config.js - module 中配置
	- plugins
		- 使用: npm 安装之后 config.js - plugins 中配置
